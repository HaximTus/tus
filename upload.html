📤 上传试卷

💡

需要运行管理脚本来上传试卷

由于网站部署在 GitHub Pages（纯静态），上传试卷需要通过管理脚本操作。

在项目目录打开终端，运行：

node scripts/manage.js

上传试卷步骤：

  1. 1
    准备 PDF 文件

    将试卷 PDF 放在 assets/papers/ 目录下

  2. 2
    运行管理脚本

    终端运行 node scripts/manage.js，选择「添加试卷」

  3. 3
    填写信息

    按提示填写试卷标题、年份、学期等信息

  4. 4
    生成并部署

    脚本会自动更新数据文件，你也可以选择一键提交到 GitHub

📌 后续版本会添加在线上传功能，届时同学们可以直接在网页上提交试卷！
