📖 新建科目

💡

需要运行管理脚本来添加科目

由于网站部署在 GitHub Pages（纯静态），添加科目需要通过管理脚本操作。

在项目目录打开终端，运行：

node scripts/manage.js

管理脚本支持的功能：

  • ✅ 列出所有科目 — 查看当前已有的科目和试卷数量
  • ✅ 新建科目 — 添加课程名称和授课教师
  • ✅ 添加试卷 — 选择科目，填写信息，拷贝 PDF 到指定目录
  • ✅ 生成数据文件 — 自动生成 JSON 数据供网站使用
  • ✅ 部署到 GitHub — 一键提交并推送更新
